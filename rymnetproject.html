Rymnet HRMS

Objective

Provide Rymnet customers with a cloud-based HRMS application that centralizes HR activities, streamlines administration, and facilitates efficient communication and collaboration between employees and administrators, ultimately meeting all human resource management needs effectively.


Features

Keep Login: Users remain logged in until they choose to logout, eliminating the need to repeatedly go through the login page.

GPS Detection: Users can only clock in / out inside the registered location radius

Beacon Detection: Users can only clock in / out using registered beacon

Wifi Detection: Users can only clock in / out using registered Wifi
