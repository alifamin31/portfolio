Infinity Logistics & Transport's Projects




Driver App

Objective

To replace paper usage. Users can conduct equipment condition checklists using this app.


Features

Remember Me: Even after logout, the password and username will autofill on the login page (if Remember Me is ticked before the first login).

Keep Login: Users remain logged in until they choose to logout, eliminating the need to repeatedly go through the login page.

Geofencing: Users can only perform tasks when they are inside the geofenced area.

Capture Image: Users can capture images of equipment parts using the app.

Image Gallery: Users can select images from the gallery to complete tasks.

Connectivity: Users can still use and complete tasks even without an internet connection.

Local Storage: If users don't have an internet connection when performing tasks, data will be stored locally.

Auto Send Data: When an internet connection is available, data stored locally will be sent to the database automatically.

BizCard App

Objective

To replace paper usage. Users can facilitate networking by sharing a QR Code for easy connection.


Features

Remember Me: Even after logout, the password and username will autofill on the login page (if Remember Me is ticked before the first login).

Connectivity: Users can continue networking activities seamlessly.
